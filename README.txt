Proposta:
Para esta solução fiz da maneira mais rápida, sendo assim criei um arquivo com o nome dos autores (busquei em um site na internet), e iniciei o projeto a partir dele.
A ideia foi um script em python onde, temos um prompt para ler um número inteiro, e a partir deste número correr a lista lendo as linhas até este número (exemplo, número 5 mostrará os 5 primeiros autores da lista). Após obter este resultado, implementei as outras solicitações do exercicio.
É possivel fazer algo mais elaborado, mas não iria ter tempo. A ideia seria algo como fazer uma base de dados e carregar os autores e colocar esta base em um container, nisto o script em python se conectaria a base de dados e traria o resultado de acordo com as condições solicitadas

Execução:
Para executar basta dar 2 cliques no programa desfio.py e inserir o número de autores conforme prompt. Também é possivel clicando com o botão direito no desafio.py->edit with idle->selecione o seu idle ele irá abrir o codigo do programa, basta clicar em executar que terá o mesmo resultado.
